## 重みから評価をする方法

* とりあえず現状のアルゴリズムでいくつかの任意の人の重みを取得する
* 重みは自分のオリジナルの方法で実行

| どんな人か | ユーザースコア | 重み |
| ---- | ---------- | ---- |
| あまり活動的でない | 0.0698 | 51.036 |
| 普通 | 6.4155 | 50.0 |
| よく書いている人 | 10.758 | 49.309 |

* ユーザースコア＝(要素n\*オフセットn+要素n1\*オフセットn1)/100
* **50前後を平均として評価したい**
